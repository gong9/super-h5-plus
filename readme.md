# super-h5 【可视化搭建 h5】

## 关于依赖安装
可先cd 到packages下的个个子项目 运行 npm i
然后退回项目根目录 yarn
再yarn start

## 关于后面工作进度
- 因方向转变，后续这个项目可能更新频率降低

由三个项目组成

- super-editor 【编辑器】
- super-show 【h5 展示】
- super-template 【组件物料】
  - react
  - vue
  - 其他

预览【本地】

yarn
yarn start
http://localhost:8000 【编辑器】
