# super-h5 【可视化搭建h5】

由三个项目组成

- super-editor    【编辑器】
- super-show      【h5展示】
- super-template  【组件物料】
  - react
  - vue
  - 其他


预览【本地】

yarn
yarn start
http://localhost:8000 【编辑器】
